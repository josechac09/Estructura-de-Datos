# Fibonacci Recursivo vs Iterativo

## Descripción
Programa en Python que calcula la sucesión de Fibonacci usando dos métodos:
* Método recursivo
* Método iterativo
El programa es interactivo, permite ingresar un valor `n`, muestra el resultado, compara los tiempos de ejecución y analiza la eficiencia de cada método.


## Objetivo
Desarrollar una solución interactiva del problema de Fibonacci en Python, identificar las ventajas de cada método y comparar su rendimiento.


## Funcionalidades
* Solicita el valor de `n`.
* Calcula Fibonacci con recursividad e iteración.
* Muestra el tiempo de ejecución de cada método.
* Permite observar el proceso paso a paso.
* Compara cuál método es más eficiente.

## Comparación de métodos
-- Recursivo
* Fácil de entender.
* Sigue la definición matemática.
* Más lento para valores grandes.
* Mayor consumo de memoria.

  --Iterativo
* Más rápido.
* Menor uso de memoria.
* Más eficiente.
* Menos intuitivo.

## Conclusión
El método recursivo es más simple conceptualmente, mientras que el método iterativo es más eficiente en tiempo y recursos.
