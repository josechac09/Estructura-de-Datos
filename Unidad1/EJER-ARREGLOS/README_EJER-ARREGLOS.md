Este programa en Python permite gestionar las ventas mensuales de una tienda que cuenta con tres departamentos: Ropa, Deportes y Juguetería. El sistema trabaja con datos previamente cargados que representan las ventas de cada departamento durante los doce meses del año. Para organizar la información se utilizan listas: una para los meses, otra para los departamentos y una matriz donde cada fila corresponde a un mes y cada columna a un departamento. El programa funciona mediante un menú interactivo que se repite continuamente hasta que el usuario decide salir, permitiendo realizar distintas operaciones sobre los datos de ventas.

El método mostrar_tabla se encarga de imprimir en pantalla una tabla ordenada con los meses y las ventas de cada departamento, recorriendo las listas y la matriz para mostrar la información de forma clara. El método insertar_venta solicita al usuario el mes, el departamento y el monto de la venta, busca la posición correspondiente dentro de la matriz de ventas y reemplaza el valor existente por el nuevo monto ingresado. El método buscar_venta pide al usuario un mes y un departamento y muestra el valor de la venta almacenada en esa posición específica de la matriz. Por último, el método eliminar_venta también solicita un mes y un departamento, pero en lugar de mostrar el valor, lo elimina asignándole el valor cero. Todas estas funciones se ejecutan desde el menú principal, que permite al usuario seleccionar la opción deseada y gestionar las ventas de manera sencilla desde la consola.
